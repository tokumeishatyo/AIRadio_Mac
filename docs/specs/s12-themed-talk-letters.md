# S12 — 季節コンテキスト + テーマプール付きフリートーク + お便りコーナー

## 1. 概要
連続放送（S13）に向けたコーナーの多様化。番組構成はまだ固定 1 周のまま、コーナーの中身を拡充する。

1. **季節コンテキストの注入**: 台本生成プロンプトに現在の日付と季節を渡し、「6 月に春めいて三寒四温」の
   ような**季節ズレを修正**する。
2. **テーマプール付きフリートーク**: コーナーにテーマのプール（お酒/料理/スイーツ/旅行/スポーツ/音楽/
   映画…）を持たせ、準備のたびに**ランダムに 1 つ選ぶ**。
3. **お便りコーナー**: LLM が架空のリスナーのお便り（ラジオネーム + 本文）を生成 → DJ 二人が読み上げて
   感想を話す → お便りに合うリクエスト曲を 1 曲流す（生成ロジックは同じ、プロンプト変種）。

## 2. スコープ

**in**:
- Core `SeasonPhrases`: 月 → 季節の言い回し（1 月=冬・正月明け / … / 6 月=梅雨 / … / 12 月=年の瀬）と
  `dateContext(date:timeZone:)`（例: 「今日は 6 月 12 日、梅雨の時期です。」）。境界はコードでハードコード
- `DialogueScriptGenerator` 拡張:
  - `theme` を引数化（テンプレ固定値ではなく選択済みテーマを渡す）
  - `dateContext` をプロンプトに注入 + 制約「季節や時候の話は、この日付・季節に合わせる」
- `CornerTemplate` 拡張: `themePool: [String]`（空ならこれまでどおり `theme` 固定）と
  `format: free_talk | letter`
- `CornerEngine.prepare`:
  - テーマ選択: `themePool` からランダム（乱数は注入可能にしてテストは決定論的）。選択テーマは
    選曲コンテキスト・台本プロンプトの両方に使う
  - letter フォーマット: ①お便り生成（LLM）→ ②リクエスト曲選定（お便り内容を選曲コンテキストに、
    プレフライトは従来どおり）→ ③台本生成（お便りの読み上げ + 感想 + 曲振り）
- Core `ListenerLetter`（ラジオネーム + 本文）+ 生成プロンプト + パース
  （出力契約: 1 行目 = ラジオネーム、2 行目以降 = 本文。装飾は除去、本文空なら
  `E-LLM-SCRIPT-PARSE-FAILED-001`）
- config:
  - corners.yaml: `free_talk` に `themes:`（プール）、新コーナー `letter`（format: letter、同プール可）
  - program.yaml: OP → song → talk(free_talk) → **talk(letter)** → news → ED
- テスト: SeasonPhrases 全月 / プロンプトの日付・季節・テーマ注入 / お便りパース /
  letter 準備の 3 段 LLM 呼び出し / テーマランダム選択（注入乱数）/ ローダ（themes・format）

**out（後続）**:
- コーナー数駆動の番組生成・ローリング準備・ED ボタン・番組の長さ UI → S13
- ゲストコーナー → S14 / アーティスト特集 → S15

## 3. お便りコーナーの台本構造

```
DJ A: 「ここでリスナーの皆さんからのお便りを紹介するのだ。ラジオネーム◯◯さんから…」（本文読み上げ）
DJ A/B: お便りへの感想・脱線（テーマ + 季節に沿う）
締め: 「◯◯さんからのリクエスト曲、△△で「□□」」（プレフライト済みの実曲）
```
- お便り本文は台本プロンプトに**全文**渡し、読み上げパートに自然に織り込ませる。
- リクエスト曲はお便りの内容・テーマを選曲コンテキストにする（fallback は従来どおり）。

## 4. 季節区分（SeasonPhrases、ハードコード）

| 月 | 言い回し |
|---|---|
| 1 | 冬、正月明け | 2 | 冬の終わり | 3 | 春の始まり | 4 | 春 |
| 5 | 初夏、新緑の季節 | 6 | 梅雨の時期 | 7 | 盛夏 | 8 | 真夏、残暑 |
| 9 | 初秋 | 10 | 秋 | 11 | 晩秋 | 12 | 冬、年の瀬 |

## 5. 受け入れ条件
- `swift build` / `swift test` 全グリーン
- 放送のフリートークが**プールから選ばれたテーマ**で行われ、**季節の言及が 6 月（梅雨）に合っている**
  （ユーザー確認）
- お便りコーナー: ラジオネーム付きのお便り読み上げ → 感想 → リクエスト曲、が自然に流れる（ユーザー確認）
- お便り→ニュースの順で番組が進む
- エラーコード追加なし
